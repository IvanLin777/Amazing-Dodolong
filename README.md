## 🎯 進化系統

| 階段 | 名稱 | 分數 | 能力 |
|:----:|:----:|:----:|:-----|
| 1 | 🟢 綠毛蟲 | 0-9 | 吃食物時清除周圍9宮格敵人，敵人暫停1秒 |
| 2 | 🔵 逗逗龍 | 10-24 | 可穿透敵人、可摧毀障礙物(+2分) |
| 3 | ⚡ 烈空座 | 25+ | 可穿牆(冷卻5秒)，擁有強大技能 |

## 🎮 操作說明

| 按鍵 | 功能 |
|:----:|:-----|
| W / A / S / D | 移動方向 |
| E | 龍衝 - 消除周圍5格內敵人 (逗逗龍/烈空座) |
| 空白鍵 | 龍波動 - 清除所有敵人 (烈空座) |
| Q | 暴風驅散 - 無敵3秒 (烈空座) |
| P | 暫停遊戲 |

## ⚔️ 技能詳情

### 綠毛蟲技能
- **吃食物** → 敵人暫停1秒 + 清除9宮格內所有敵人

### 逗逗龍技能
- **E鍵龍衝** → 消除周圍5格內所有敵人 (冷卻3秒)

### 烈空座技能
- **空白鍵龍波動** → 清除畫面所有敵人 (冷卻3秒)
- **Q鍵暴風驅散** → 無敵狀態3秒
- **穿牆** → 穿越邊界 (冷卻5秒)

## 💀 困難設計

- 每 **3秒** 生成障礙物 (█)
- 每 **5秒** 生成敵人 (✸)
- 敵人撞到障礙物會消失
- 每吃 **3個食物** 速度加快
